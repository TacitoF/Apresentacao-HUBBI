Adesão ao Ambiente HubBI

Fluxo de Solicitação e Configuração

1

Formalização

O Gestor Setorial deve formalizar a adesão via SEI. É necessário utilizar o modelo de adesão oficial disponível.

Apenas Gestor Setorial
2

Pré-requisitos

Possuir login (conta e senha) no SEI ou e-mail expresso.pe.go.br.

Login obrigatório na plataforma HubBI (1ª vez)
3

Permissões

O Gestor Setorial concede a permissão de Desenvolvedor diretamente pela interface de administração do portal HubBI.

4

Pós-Adesão

O Gestor define os usuários finais e relatórios, aplicando o Controle de Acesso Lógico (CAL).

Atenção aos Requisitos de Conta

Para assumir o papel de Gestor Setorial ou Desenvolvedor, é mandatório que o usuário tenha realizado login na plataforma HubBI pelo menos uma vez previamente. Isso garante que o perfil de usuário exista na base de dados para atribuição de permissões.
